#-----断言---（assert）-----了解
    所谓的断言，指的是程序在编写过程中，确定代码执行到某行之后数据一定是吗某个期待的内容。
    
    对于断言在默认情况下是不会使用的，只有在程序使用了一些配置参数才可以使用。
